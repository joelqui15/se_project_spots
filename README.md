# Spots

Online media app

## Description

An interactive social media web application where users can add and remove photos, like photos of other users, and make personal adjustments to their profile.

## TECH STACK

- HTML
- CSS
- RESPONSIVE DESIGN
- BEM
- GIT/GITHUB
- VSCODE
- FIGMA
